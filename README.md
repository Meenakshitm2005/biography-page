# Biography Page

A simple and responsive Biography Page created using HTML and CSS.

## About the Project

This project is a personal biography webpage that showcases:

* Personal introduction
* Educational background
* Skills
* Hobbies and interests
* Contact information
* Profile picture

The page is styled using CSS to provide a clean and attractive user interface.

## Technologies Used

* HTML5
* CSS3

## Features

* Responsive design
* Circular profile image
* Clean card-style layout
* Organized sections for easy navigation
* Simple and beginner-friendly code structure
